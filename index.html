A rapid-paced game about spheroid with high momentum and a target (in C++11 and SFML2.1)

Presentation

Impetus Spheroid is a dynamic arcade game where you must touch a lively target with your sphere with high momentum. Gravity, time and obstacles will try to stop you, but you can rely on your bonus to overcome them. Try to blow up your score !

Getting the Game

For Windows users: simply go on the itch.io page and follow the instructions at the bottom :)

GNU/Linux fanatics: you'll have to compile from source and install libsfml but that's easy:

$ sudo apt-get install g++ make libsfml-dev
$ git clone https://github.com/Epholys/Impetus-Spheroid
$ cd Impetus-Spheroid
$ make install
$ ./Impetus-Spheroid

Hacking

You'll only need three things :

  • A C++11-compliant compiler like gcc
  • SFML 2.1 or later, available here and with some excellent setup tutorials there
  • The source and media of the game, available in a neat package at the top of this page
